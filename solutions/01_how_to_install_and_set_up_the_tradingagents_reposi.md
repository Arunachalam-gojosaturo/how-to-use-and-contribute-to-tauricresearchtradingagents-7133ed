# How to install and set up the TradingAgents repository for local development

_Install required dependencies and clone the repository for local development_

## Steps

1. Open a terminal and run `git clone https://github.com/TauricResearch/TradingAgents.git` to clone the repository
2. Navigate into the cloned repository using `cd TradingAgents`
3. Create a new virtual environment using `python3 -m venv venv`
4. Activate the virtual environment using `source venv/bin/activate` on Linux/Mac or `venv\Scripts\activate` on Windows
5. Install required dependencies using `pip install -r requirements.txt`

## Pitfalls

- Failing to activate the virtual environment before installing dependencies
