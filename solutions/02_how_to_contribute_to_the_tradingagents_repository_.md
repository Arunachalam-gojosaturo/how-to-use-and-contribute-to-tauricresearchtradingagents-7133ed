# How to contribute to the TradingAgents repository by fixing a bug

_Fork the repository, fix the bug, and submit a pull request_

## Steps

1. Fork the TradingAgents repository using the GitHub web interface
2. Clone the forked repository using `git clone https://github.com/your-username/TradingAgents.git`
3. Create a new branch using `git checkout -b fix/bug-name`
4. Make the necessary changes to fix the bug
5. Commit the changes using `git add .` and `git commit -m 'Fix bug: brief description'`

## Pitfalls

- Not creating a new branch for the bug fix, which can cause conflicts with the main branch
